blog创建流程

第一次创建的blog半个月后不能使用hexo， 为了防止情况再次发生时忘记怎么initblog，创建此备忘录

本地创建过程

  1. 在./hexo文件夹 git bash here
  2. 初始化bloghexo init blog
  3. 本地部署hexo s
  4. 创建新文章 hexo n "xxx"
  5. 更新生成hexo g
  6. hexo s

github部署

  1. 先删除github的仓库（感觉也可以直接更新新的地址）

  2. 新blog的_config.yml文件中修改deploy配置.

  3. 安装hexo部署 npm install --save hexo-deployer-git

  4. 开始部署 hexo d 由于之前已经设置过git相关的，所以没有弹出账号密码的设置

  5. 如果遇到

    是网络问题 更换梯子就行

  6. 之后等待1-2分钟之后访问即可

更改主题以及Site

  1. npm install hexo-theme-next

  2. #Site

  3. hexo clean && hexo g

  4. hexo s

  5. hexo d
